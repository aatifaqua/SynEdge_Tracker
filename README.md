## ✨ About SynEdge

SynEdge replaces your system's default volume panel with highly customizable, modern aesthetics. Built with performance and privacy at its core, the app operates strictly on a **100% offline, zero-data-collection architecture**.

## 🛠️ How to Use This Repository

We use GitHub Issues strictly for actionable development tracking. To help keep this repository clean and professional:

1. **Search First:** Check the [Issues tab](../../issues) to ensure your bug or feature hasn't already been reported.
2. **Use the Templates:** When creating a new issue, choose either the **Bug Report** or **Feature Request** template.
3. **Be Specific:** Include your device model, Android version, and clear steps to reproduce any bugs.
